# MyTargetClient

## Installation

**Mac OS:**

To install Chrome Driver(http://chromedriver.storage.googleapis.com/index.html) type in terminal:
```
brew install chromedriver
```
To run demo example go to project directory and type in terminal:
```
ruby init.rb
```

## Tests
```
rspec spec
```
